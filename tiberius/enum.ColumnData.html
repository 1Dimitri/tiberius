A container of a value that can be represented as a TDS value.

Variants

None

A missing value.

I8(i8)

8-bit integer, signed.

I16(i16)

16-bit integer, signed.

I32(i32)

32-bit integer, signed.

I64(i64)

64-bit integer, signed.

F32(f32)

32-bit floating point number.

F64(f64)

64-bit floating point number.

Bit(bool)

Boolean.

String(Cow<'a, str>)

A string value.

Guid(Uuid)

A Guid (UUID) value.

Binary(Cow<'a, [u8]>)

Binary data.

Numeric(Numeric)

Numeric value (a decimal).

Xml(Cow<'a, XmlData>)

XML data.

DateTime(DateTime)

DateTime value.

SmallDateTime(SmallDateTime)

A small DateTime value.

Time(Time)

Time value.

Date(Date)

Date value.

DateTime2(DateTime2)

DateTime2 value.

DateTimeOffset(DateTimeOffset)
